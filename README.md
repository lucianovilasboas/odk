# odk

Este repositório contém scripts para preparar a instalação do ODK Central.

Instruções de instalação detalhadas: [INSTALACAO.md](INSTALACAO.md)

Arquivos úteis:
- `install_odk_novo.sh` — script recomendado para preparar o servidor e instalar Docker/ODK.
- `install_odk.sh` — versão simples/antiga do instalador.
- `create_useradmin_odk.sh` — cria usuário e opcionalmente promove a admin.
